### Test
to deploy locally:
```
jekyll serve --livereload
```

note: livereload doesn't account for changes to _config

### Add new pages
For a new page available at url/``page-name``, create one of the following in _pages:

- ``page-name``.md, or
- ``page-name``.html, or
- ``page-name`` /index.[md, html]
